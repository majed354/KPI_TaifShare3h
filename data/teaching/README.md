# Teaching Data (for Student/Faculty Ratio)

هذا المجلد يُستخدم لحساب مؤشر **نسبة الطلاب/هيئة التدريس** تلقائيًا من التدريس الفعلي.

## الملفات المطلوبة

1. `../new_all_plans.csv`
   - ربط رمز المقرر بالبرنامج والدرجة.
2. `years/<year>.json`
   - ملف التدريس الفعلي السنوي (مثل `1447.json`).
3. `../faculty.csv`
   - فهرس الأعضاء السنوي (للـ fallback عند غياب تدريس فعلي لبعض الأعضاء).

## التحديث السنوي

1. أضف ملف السنة الجديدة داخل `years/` (مثال `1448.json`).
2. حدّث `new_all_plans.csv` إذا ظهرت رموز مقررات جديدة.
3. حدّث `data.csv` لبيانات الطلاب في السنة نفسها.

بعد ذلك سيحسب الموقع النسبة تلقائيًا عند التحميل.

## 🧮 احتساب نسبة الطلاب/هيئة التدريس (تلقائي)

تم اعتماد الاحتساب من **التدريس الفعلي** بدل الإدخال اليدوي فقط:

- المصدر الأساسي: `data/teaching/years/<year>.json`
- ربط المقرر بالبرنامج: `data/new_all_plans.csv`
- fallback للأعضاء النشطين غير الموجود لهم تدريس فعلي: `data/faculty.csv`

### آلية التوزيع
- يُوزَّع نصاب العضو بين البرامج حسب عبئه التدريسي الفعلي في السنة.
- إذا كان العضو يدرّس في برنامجين بالتساوي -> يُحتسب تقريبًا `0.5` لكل برنامج.
- المتعاون يُحتسب كـ FTE جزئي (افتراضيًا `0.5` ويمكن تعديلها من `js/app.js`).

### التحديث للفصول القادمة
1. أضف/حدّث ملف السنة في: `data/teaching/years/` (مثال: `1448.json`)
2. تأكد من وجود مقررات السنة في: `data/new_all_plans.csv`
3. حدّث `data/data.csv` لبيانات الطلاب للسنة نفسها
4. افتح الموقع؛ النسبة تُحسب تلقائيًا بدون تعبئة `faculty_total` يدويًا

## 🔗 ربط الموقع بـ Google Sheets

### ربط استطلاع الخريجين (المؤشرات 1 و5 و6 و7)
1. في Google Sheets افتح تبويب **ردود النموذج** الخاص باستطلاع الخريجين.
2. انشره كـ CSV:
   **ملف** → **مشاركة** → **نشر على الويب** → اختر التبويب المطلوب → **CSV**.
3. انسخ رابط CSV المنشور.
4. في [index.html](/Users/majd/Desktop/codex/KPI_TaifShare3h-main/index.html) عدّل الإعداد:
```javascript
window.KPI_CONFIG = {
    graduatesSurveySheetUrl: '', // اختياري (شيت واحد لكل الدرجات)
    graduatesSurveyBachelorSheetUrl: 'رابط شيت البكالوريوس',
    graduatesSurveyPostgradSheetUrl: 'رابط شيت الدراسات العليا'
};
```
5. احفظ وارفع التحديث.

### وضع التشغيل المقترح
- إذا لديك شيتان منفصلان: املأ `graduatesSurveyBachelorSheetUrl` و `graduatesSurveyPostgradSheetUrl` واترك `graduatesSurveySheetUrl` فارغًا.
- إذا لديك شيت واحد فقط: املأ `graduatesSurveySheetUrl` فقط.

### الأعمدة المطلوبة في شيت الاستطلاع
النظام يقرأ هذه الحقول من الشيت (يكفي وجودها بنفس المعنى حتى لو اختلفت الصياغة قليلًا):
- اسم البرنامج
- سنة التخرج
- التقييم العام لجودة التعلم في البرنامج
- الوضع الحالي بعد التخرج
- درجة الاختبارات الوطنية/المهنية
- تقييم جهة التوظيف/الرئيس المباشر

### كيف تُحسب المؤشرات تلقائيًا
- `تقويم الطالب لجودة خبرات التعلم` ← متوسط تقييم جودة التعلم (من 5)
- `مستوى أداء الطالب` ← متوسط درجات الاختبارات (كنسبة %)
- `توظيف الخريجين` ← نسبة الحالات المصنفة "موظف" من الحالات ذات الوضع المعروف
- `تقويم جهات التوظيف` ← متوسط تقييم جهة التوظيف (من 5)

### ملاحظات مهمة
- إذا كان رابط الشيت فارغًا فلن يتم تحديث هذه المؤشرات من Google Sheets.
- عند إضافة ردود جديدة في النموذج، تظهر القيم الجديدة في الموقع تلقائيًا بعد إعادة تحميل الصفحة.
- يجب أن يكون الشيت متاحًا للقراءة عبر الرابط (Published CSV أو Anyone with link).
- عند استخدام شيت البكالوريوس: التطبيق يطبّق نتائجه على برامج `بكالوريوس` فقط.
- عند استخدام شيت الدراسات العليا: التطبيق يطبّق نتائجه على برامج `الماجستير` و`دكتوراه` فقط.
